AstroTaverna

To install the AstroTaverna plugin to Taverna:

  • Download and install Taverna 2.4
  • Start Taverna
  • Add a plugin site: http://wf4ever.github.com/astrotaverna/
  • Restart Taverna
  • The VO services perspective should now appear together with Astro tools under Available Services

For more information and support, see the AstroTaverna homepage or contact us at astrotaverna-users@iaa.es.
