AstroTaverna

To install the AstroTaverna plugin to Taverna:

  • Download and install Taverna 2.4
  • Start Taverna
  • Add a plugin site: http://wf4ever.github.com/astrotaverna/
  • Restart Taverna
  • The VO services perspective should now appear together with variouys local tools under Available Services

For more information, watch our video tutorial or go to the Astrotaverna starter pack that includes a compilation of examples with example input data and descriptions.
